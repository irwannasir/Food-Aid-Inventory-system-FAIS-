# Food-Aid-Inventory-System-FAIS-
FAIS is a Java command line app that automates tracking donated food to reduce waste. It replaces manual logging with data structures (Queues) and search algorithms to instantly sort inventory into active or expired categories. This ensures accurate stock levels and speeds up distribution for aid volunteers.
## 🛠️ How to Run the Application

Anyone can download and run this system locally by following these steps:

1. **Prerequisites:** Make sure you have the Java Development Kit (JDK 8 or higher) installed on your system.
2. **Download the Repository:** Click the green **`Code`** button at the top of this page and select **Download ZIP** (then extract the files).
3. **Compile the System:** Open your Terminal (Mac/Linux) or Command Prompt (Windows) inside the project folder and compile the main application file:
```bash
   javac FoodItemApp.java
